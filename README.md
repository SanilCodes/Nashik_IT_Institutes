# Nashik_IT_Institutes
In this Project I gathered data of Nashik Institutes and made a dashboard using Power BI

I collected and analyzed data related to major IT training institutes in Nashik using online platforms such as Justdial and Sulekha. The data included institute names, locations, courses offered, course duration, fees, ratings, contact details, and other relevant information.

After gathering the data from multiple sources, I cleaned and organized it in CSV format by removing duplicates, handling missing values, and standardizing entries for consistency. This structured dataset was then used for analysis and visualization.

Using the processed data, I created an interactive dashboard to present meaningful insights such as:

Distribution of institutes across Nashik
Popular IT courses offered
Course fee comparisons
Institute ratings and reviews
Duration-wise course analysis
